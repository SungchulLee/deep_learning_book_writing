# List Method sort

```python
a = [-9, 1, 8, 2, -7, 3, 6, 4, 5]
a.sort()
a
```

**Output:**
```
[-9, -7, 1, 2, 3, 4, 5, 6, 8]
```

```python
a = [-9, 1, 8, 2, -7, 3, 6, 4, 5]
a.sort(reverse=True)
a
```

**Output:**
```
[8, 6, 5, 4, 3, 2, 1, -7, -9]
```
