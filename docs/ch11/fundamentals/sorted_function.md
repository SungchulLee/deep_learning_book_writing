# Built-in sorted


```python
if 0:
    a = [9, 1, 8, 2, 7, 3, 6, 4, 5]
elif 0:
    a = (9, 1, 8, 2, 7, 3, 6, 4, 5)
elif 0:
    a = {'name': 'Jake', 'job': 'Programming', 'age': '29', 'os': 'Mac'}
elif 1:
    a = {'name', 'job', 'age', 'os'} 
elif 0:
    a = [(3,34), (2,35), (4,30), (3,33)]
```


```python
sorted(a)
```

**Output:**
```
['age', 'job', 'name', 'os']
```


```python
sorted(a, reverse=True)
```

**Output:**
```
[9, 8, 7, 6, 5, 4, 3, 2, 1]
```
