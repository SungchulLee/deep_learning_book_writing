# Level 1: Basics

## Overview
This directory contains introductory tutorials covering fundamental concepts of PyTorch and logistic regression.

## Files

### 01_introduction.py
- **Difficulty**: ⭐☆☆☆☆
- **Time**: 30 minutes
- **Topics**: PyTorch tensors, autograd, gradient descent
- **Prerequisites**: Basic Python

### 02_simple_binary_classification.py
- **Difficulty**: ⭐⭐☆☆☆
- **Time**: 45 minutes
- **Topics**: Binary classification, sigmoid, training loop
- **Prerequisites**: Completed 01_introduction.py

### 03_with_sklearn_data.py
- **Difficulty**: ⭐⭐☆☆☆
- **Time**: 1 hour
- **Topics**: Real datasets, preprocessing, evaluation metrics
- **Prerequisites**: Completed 02_simple_binary_classification.py

### 04_gradient_descent_with_BCE.py
- **Difficulty**: ⭐⭐☆☆☆
- **Time**: 45 minutes
- **Topics**: Detailed BCE loss implementation, shape annotations
- **Prerequisites**: Understanding of binary classification

### 05_gradient_descent_with_BCEWithLogitsLoss.py
- **Difficulty**: ⭐⭐☆☆☆
- **Time**: 45 minutes
- **Topics**: Numerical stability, BCEWithLogitsLoss
- **Prerequisites**: Completed 04

### 06_gradient_descent_advanced.py
- **Difficulty**: ⭐⭐⭐☆☆
- **Time**: 1 hour
- **Topics**: Advanced gradient descent patterns
- **Prerequisites**: Understanding of optimization

### 07_sklearn_comparison.py
- **Difficulty**: ⭐⭐☆☆☆
- **Time**: 45 minutes
- **Topics**: PyTorch vs scikit-learn comparison
- **Prerequisites**: Familiarity with sklearn

### 08_imdb_text_classification.py
- **Difficulty**: ⭐⭐⭐☆☆
- **Time**: 1.5 hours
- **Topics**: Text classification, embeddings, NLP basics
- **Prerequisites**: Completed basics 01-03

## Learning Path
**Recommended order**: 01 → 02 → 03 → 04 → 05

**Optional deep dives**: 06, 07, 08 (can be done in any order after completing 01-05)

## Expected Outcomes
After completing this level, you should be able to:
- Create and manipulate PyTorch tensors
- Understand automatic differentiation
- Implement basic logistic regression
- Train models using gradient descent
- Evaluate binary classifiers
- Compare different loss functions
- Work with text data basics
