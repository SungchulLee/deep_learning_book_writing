# 29.7.3 Social Network Analysis

## Overview
GNNs model social networks for community detection, influence prediction, information cascade modeling, and bot/fake account detection.

## Tasks
- **Influence maximization**: Identify nodes that maximize information spread
- **Cascade prediction**: Predict how information propagates
- **Bot detection**: Classify accounts using network structure + behavior features
- **Link prediction**: Predict future connections (friend recommendations)

## Challenges
Scale (billions of edges), dynamic evolution, heterogeneous interactions, privacy constraints.
