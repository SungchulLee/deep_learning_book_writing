# Big-O Notation

**Big-O Notation** is an important concept in algorithm design and analysis.

$$f(n) = O(g(n)) \iff \exists\, c > 0,\, n_0 > 0 : f(n) \leq c \cdot g(n) \;\forall n \geq n_0$$
